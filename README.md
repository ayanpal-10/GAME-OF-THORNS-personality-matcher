# ⚔️ Game of Thrones Personality Matcher

A fun NLP project that recommends the most similar **Game of Thrones** character based on dialogue similarity.

---

## 📌 Features

- Character similarity recommendation
- NLP using Bag-of-Words
- t-SNE dimensionality reduction
- Interactive Streamlit web application
- Simple and easy-to-use interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- Requests

---

## 📂 Dataset

**Game of Thrones Dialogue Dataset**

Source:
https://www.kaggle.com/datasets/mathurinache/game-of-thrones-data

Dataset File:
`script-bag-of-words.json.zip`

---

## ⚙️ Machine Learning Techniques

- Bag-of-Words
- CountVectorizer
- t-SNE
- Euclidean Distance Similarity

---

## 🚀 Project Workflow

1. Load dialogue dataset
2. Preprocess text
3. Convert text using Bag-of-Words
4. Apply t-SNE
5. Calculate character similarity
6. Recommend the closest character
7. Display the result using Streamlit
