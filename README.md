# GAME-OF-THORNS-personality-matcher




A fun NLP project that recommends the most similar Game of Thrones character based on dialogue.

## Features

- NLP using Bag-of-Words
- t-SNE dimensionality reduction
- Similar character recommendation
- Streamlit web app

## Dataset

Game of Thrones Dataset (Kaggle)

## Technologies

- Python
- Streamlit
- scikit-learn
- NumPy
- Pandas
